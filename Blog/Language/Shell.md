
## 特殊变量

- $0 脚本名
- $n 位置参数, 例如$1表示第一个参数,如果是两位数需要加括号${10}
- $# 参数个数,传递给脚本或函数的参数总数
- $? 退出状态码,0表示成功,非零表示失败或出错
- $* 所有参数(单字符串),将所有参数视为一个单一的字符串,例如"$1 $2 $3"
- $@ 所有参数(独立字符串),将所有参数视为独立的字符串数组,例如"$1" "$2",在带空格的参数遍历中更安全
- $$ 当前PID,当前Shell进程的进程ID号(Process ID)
- $! 后台PID,上一个在后台运行的进程PID
- $_ 上一个参数,上一条命令的最后一个参数

## if条件操作符

在`if [ condition ]`,或`if [[ condition ]]`中使用的符号:

文件与目录:
操作符,用法示例,说明 (如果条件成立则返回 True)
- -e,[ -e file ],文件或目录存在 (Exist)
- -f,[ -f file ],存在且是一个普通文件 (File)
- -d,[ -d dir ],存在且是一个目录 (Directory)
- -s,[ -s file ],存在且大小大于 0 (即文件非空) (Size)
- -r,[ -r file ],存在且可读 (Readable)
- -w,[ -w file ],存在且可写 (Writable)
- -x,[ -x file ],存在且可执行 (Executable)
- -L,[ -L file ],存在且是一个软链接 (Link)
- -c,[ -c file ],存在且是一个字符设备文件
- -b,[ -b file ],存在且是一个块设备文件

整数比较测试:
- 操作符,英文全称,用法示例,说明
- -eq,equal,[ $a -eq $b ],等于
- -ne,not equal,[ $a -ne $b ],不等于
- -gt,greater than,[ $a -gt $b ],大于
- -ge,greater or equal,[ $a -ge $b ],大于或等于
- -lt,less than,[ $a -lt $b ],小于
- -le,less or equal,[ $a -le $b ],小于或等于

字符串比较:
- = 或 ==,"[ ""$a"" == ""$b"" ]",两个字符串相同
- !=,"[ ""$a"" != ""$b"" ]",两个字符串不同
- -z,"[ -z ""$str"" ]",字符串长度为零 (Zero / 为空)
- -n,"[ -n ""$str"" ]",字符串长度非零 (Non-zero / 不为空)

逻辑操作符:
- 与: -a(单括号) 或 &&(双括号)
- 或: -o(单括号) 或 ||(双括号)
- 非: !
